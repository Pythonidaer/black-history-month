# Day 01 — Fred Hampton

**Live site:** [black-history-month-phi.vercel.app](https://black-history-month-phi.vercel.app/)

A tribute page for Fred Hampton (1948–1969), civil rights leader, Black Panther Party deputy chairman, and founder of the Rainbow Coalition. This website covers aspects of his life, the Rainbow Coalition, why he was targeted by COINTELPRO, the 1969 raid and assassination, and his lasting legacy.

## Run locally

```bash
cd day-01-fred-hampton && npm install && npm run dev
```

## Tech

- Vite + React
- Tailwind CSS, Flowbite, react-vertical-timeline, react-icons
- Deployed via Vercel (repo root + `vercel.json`)
